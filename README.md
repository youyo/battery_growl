# BatteryGrowl

When the MacBookAir's battery is running low, notify the growl.

## Installation

Add this line to your application's Gemfile:

    gem 'battery_growl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install battery_growl

## Usage

    $ battery_growl [num]
